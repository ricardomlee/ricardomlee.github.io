n900 安装 lirc

apt-get install lirc

遥控配置文件示例 air.conf

文件来源：suikammd/AirConditioner-Homekit

配置 lircd.conf

  • 第一种方法：直接将air.config内容复制到 /etc/lirc/lircd.conf
  • 第二种方法：创建 /etc/lirc/lircd.conf.d 文件夹，将air.conf放入并更名为air.lircd.conf
    /etc/lirc/lircd.conf中会自动/手动添加 include "/etc/lirc/lircd.conf.d/air.lircd.conf"

启动 lirc

  • 启动：/etc/init.d/lirc start
  • 关闭：/etc/init.d/lirc stop
  • 重启：/etc/init.d/lirc restart

irsend 命令

  • 发送遥控信号： irsend SEND_ONCE air OFF
  • 列出所有信号名： irsend list "" ""
